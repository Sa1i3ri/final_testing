# 架构决策记录助手 (ADR Assistant)

## 项目简介

**ADR Assistant** 是一个基于 FastAPI 的应用程序，旨在通过用户提供的上下文生成架构决策记录（ADR）。它结合了高级语言模型、嵌入和向量存储技术，实现了高效的文档检索和生成。

## 功能特性

- **零样本提示**：基于用户上下文生成决策，无需预先示例。
- **少样本提示**：使用预定义示例指导决策生成。
- **检索增强生成 (RAG)**：从向量存储中检索相关 ADR 并生成决策。
- **评估功能**：支持 BLEU、ROUGE 和 METEOR 等指标评估生成结果。
